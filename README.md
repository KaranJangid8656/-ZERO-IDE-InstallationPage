## 🚀 Overview

**-ZERO IDE** makes code collaboration simple. Develop together, anywhere. This repository contains the marketing and installation landing page for the IDE, built with Next.js, React, and Tailwind CSS to feature a sleek and premium interface.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Components:** [Radix UI](https://www.radix-ui.com/) / shadcn
- **Icons:** [Lucide React](https://lucide.dev/)

## ✨ Features

- **⚡ Real-Time Collaboration:** Edit the same files simultaneously with zero lag, powered by WebSockets and CRDT technology.
- **👀 Live Cursor Tracking:** See exactly where your team members are typing with live colored cursors and identifying name tags.
- **🔗 Instant Room Sharing:** Generate a secure, unique link to instantly invite peers into your active coding session.
- **🌙 Premium Dark Mode:** A sleek, high-contrast, native dark UI tailored specifically to reduce developer eye strain.
- **📁 Multi-File Syncing:** Navigate through project directories and seamlessly keep your team on the same page (and file).
- **🚀 High Performance:** Optimized architecture engineered for minimal latency, even across diverse network connections.
- **💻 Familiar Developer Experience:** Standard code editor functionalities like syntax highlighting and clean file trees, right out-of-the-box.

## ⚙️ Getting Started

### Prerequisites
Make sure you have Node.js and a package manager like npm, pnpm, or yarn installed.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/KaranJangid8656/-ZERO-IDE-InstallationPage.git
   ```

2. Navigate into the project directory:
   ```bash
   cd -ZERO-IDE-InstallationPage/Installation-Page
   ```

3. Install dependencies:
   ```bash
   npm install
   # or
   pnpm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   # or
   pnpm dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
